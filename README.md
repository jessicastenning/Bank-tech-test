# Bank Tech Test

## User Stories
```
As a user I'd like to be able to deposit funds in my bank account.

As a user I'd like the date that I deposited funds in my bank account to be recorded.

As a user I'd like to be able to withdraw funds from my bank account.

As a user I'd like the date that I withdrew funds from my bank account to be recorded.

As a user I'd like my account balance to be updated when I make withdrawals/deposits.

As a user I'd like to review my bank statement.

As a user I'd like my bank statement to show all transactions with the date, amount withdrawn/deposited,
and the updated balance.
```
